# BODMAS Game
Este pequñeo juego es para practicar conceptos básicos de Python.
## Descripción
Este juego interactivo es para practicar la presedencia de operadores matemáticos. El programa te dará una operación matemática al azar y tendrás que poner el resultado de dicha operación, te dirá si es correcta la respuesta o no y te preguntará si deseas continuar. Además, llevará el score en un archivo .txt de los usuarios que lo utilizen. Ojalá que te guste, y si te sirve, lo puedas replicar, practicar o incluso mejorar.

## Uso
```bash
# ejecutar el programa
uv run python main.py
```
